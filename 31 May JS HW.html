Q.2
Given two number n=110, s=101. Print in console the addition, subtraction, multiplication and division of these numbers.

Solution:
  Here are codes:

      var n = 110;
      var s = 101;

      var ad = n + s;

      var sb = n - s;
      console.log(sb); // for subtraction

      var mp = n * s;
      console.log(mp); // for multiplication

      var dv = n / s;
      console.log(dv); // for division
    </script>

Q.4
Write a program to check whether a given number is within 50 to 100 or 200 to 400. If not print "not in range".

Solution:
  Here are codes: 
  
    <script>
    var a = 85;
    var b = 507;
    document.write("a = 85 <br> b = 507 <br>");
    if (a >= 50 && a <= 100) {
      document.write(a + " is within 50 to 100. <br>");
     }
    else if(a >= 200 && a <= 400){
      document.write(a + " is within 200 to 400. <br>");
     }
    else{
      document.write(a + " is not in range. <br>");
}
    if (b >= 50 && b <= 100) {
      document.write(b + " is within 50 to 100. <br>");
     }
    else if(b >= 200 && b <= 400){
      document.write(b + " is within 200 to 400. <br>");
     }
    else{
      document.write(b + " is not in range. <br>");
}
    </script>
